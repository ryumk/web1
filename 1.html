HTML


Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web apllications. Web browsers receive HTML documnets from a web server or form local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the documnet.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered pages. it provides a means to create structed documents by denoting structual semantics for text such as heading, paragraphs, lists, links, quotes and other items. HTML elemnts are delineated by tags, written using angle brackets.
